							Инструкция по эксплуатации программы
В поле введите функцию необходимо ввести математическое выражение, содержащее одну переменную x или не содержащее переменных. Список функций, которыми можно пользоваться: "+", "-", "^", "*", "/", "abs", "sin", "cos","ln","(",")". Также можно использовать константы, которые приведены в виде кнопок. Между числом и переменной, которые вы хотите перемножить обязательно использовать знак *. В записи десятичного числа необходимо использовать запятые. В поле пределы интегрирования вводится десятичное число, для записи используются запятые. При ошибке в вводе вы получится сообщение в строке для вывода “Неверный ввод!”.
Для выбора точности необходимо нажать кнопку с желаемой точностью.
Для выбора метода интегрирования необходимо выбрать метод из раскрывающегося списка. По умолчанию метод Симпсона.
Для построения графика необходимо корректно ввести функцию, пределы вводить необязательно (по умолчанию от -10 до 10), и нажать соответствующую кнопку.
Кнопка очистить стирает информацию из всех полей.
